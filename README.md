# SIFT

This is an implementation of SIFT (Scale-invariant feature transform).

<img src="images/result.jpeg" width="300px">

### Usage
First, install libraries.
```bash
$ pipenv install -d
```

Type this to run SIFT.
```bash
$ python main.py
```
